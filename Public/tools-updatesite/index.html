UI Development Toolkit for HTML5

UI Development Toolkit for HTML5 is available with SAP NetWeaver UI Extensions 1.0.

For official detailed download and installation instructions see the SAP Note 1747308 and the installation guide attached on this SAP Note.

To start developing applications with the UI Development Toolkit for HTML5 please follow these steps:

  1. Download and install the Eclipse IDE for Java EE Developers separately from Eclipse
  2. Install the Eclipse feature "Jetty - Core: servlets and webapps" separately from the Eclipse Juno update site
  3. Install the UI Development Toolkit for HTML5 as described in the installation guide.
For documentation on the latest version of the UI Development Toolkit for HTML5, see the DemoKit
